# Archived research and retired automation

Nothing in this directory is an active deployment.

| Path | Contents |
| --- | --- |
| `kalshi/ml/` | Retired ML inference, training, calibration, and paper-scaling code. |
| `kalshi/prophet/` | Retired Prophet BTC15M bot and selector research. |
| `kalshi/research/` | Historical backtests, streak studies, and ledger analysis. |
| `polymarket/mlb/` | Retired/dry MLB average-down and ML research. |
| `reports/` | Historical Kalshi, Prophet, ML, paper, and retired Polymarket ledgers. |
| `requirements/` | Dependency files for archived research only. |
| `tests/` | Tests for archived code. |
| `workflows/` | Retired GitHub Actions definitions. They are stored outside `.github/workflows/`, so GitHub cannot discover, schedule, or execute them. |

The active systems and their supported files are documented in the repository [README](../README.md). Archived workflows reference their original layout and are retained as historical records, not executable instructions.
